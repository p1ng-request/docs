## Model

### Define function to reformat text


```python
def reformat_text(text):
    sentences = []

    for sentence in text.split("."):
        if len(sentence) > 0:
            sentences.append(" ".join(wordninja.split(sentence)) + ".")

    return " ".join(sentences)
```
