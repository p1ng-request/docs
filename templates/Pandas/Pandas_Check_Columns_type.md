## Model

### Check Columns type in dataframe and create conditions


```python
for c in df.columns:
    print(f"Columns '{c}' is {df[c].dtype}.")
    if df[c].dtype == object:
        print(f"=> Do something")
    elif df[c].dtype == str:
        print(f"=> Do something")
    elif df[c].dtype == int:
        print(f"=> Do something")
    elif df[c].dtype == float:
        print(f"=> Do something")
```
