### Find asset link from path


```python
def find_naas_asset_link(file_path):
    if not file_path.startswith("/home/ftp/"):
        file_path = f"/home/ftp/{file_path}"
    asset_link = naas.asset.find(file_path)
    if asset_link:
        print("âœ… Asset link found:")
    return asset_link
```
